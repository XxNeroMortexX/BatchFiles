Batch Script - MORE


Previous Page
Next Page  

This batch command displays the contents of a file or files, one screen at a time.

Syntax

More [filename]

Where filename is the file whose contents need to be listed one screen at a time.

Example

@echo off
More C:\tp\lists.txt
Directory of C:\Program Files

Output
